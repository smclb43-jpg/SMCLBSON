AFTEM Mobil İstasyon - Android Studio Final Proje

Bu proje Android Studio içinde açılacak şekilde tamamlandı.

Ne var:
- İkonlu Android uygulama
- WebView içinde mobil üretim ekranı
- İş kartında: Müşteri, Proje, Cam Türü, Adet, m2
- İstasyonlar: Kesim, Rodaj, Temper, Basım, Robot
- PC API bağlantısı
- GitHub Actions ile debug APK üretim workflow'u

Android Studio:
1. Bu klasörü aç
2. Gradle sync bekle
3. Run veya Build > Build APK(s)

Canlı kullanım:
1. Bilgisayarda server.py çalıştır
2. Telefon ve bilgisayar aynı Wi‑Fi ağında olsun
3. Uygulamayı aç
4. API adresine http://BILGISAYAR_IP:8765 yaz
5. İstasyonu seçip devam et

Not:
- APK bu oturumda derlenmedi
- Bu paket Android Studio klasör yapısı içindir
